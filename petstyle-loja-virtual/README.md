# 🐾 Pet&Style

Pet&Style é uma landing page moderna para uma loja fictícia de produtos pet, desenvolvida com foco em design responsivo, organização de código e boas práticas de Front-End.

## 🚀 Tecnologias Utilizadas

HTML5

Tailwind CSS (via CLI)

JavaScript (opcional para interações)

Node.js (para build do Tailwind)

## 🎨 Funcionalidades

Layout responsivo

Gradientes modernos

Seção de produtos

Menu com interação

Seção de contato

Estilização com Tailwind Utility-First

    📂 Estrutura do Projeto
            petstyle-loja-virtual/
            │
            ├── index.html
            ├── package.json
            ├── tailwind.config.js
            │
            └── src/
                ├── input.css
                └── output.css
⚙️ Como rodar o projeto

1️⃣ Instalar dependências
        
        npm install
2️⃣ Rodar o Tailwind em modo watch

    npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
3️⃣ Abrir o index.html no navegador
