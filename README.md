## Projetos

### 1. Controle de Gastos
Um sistema simples para registrar despesas e receitas, ajudando a controlar o orçamento pessoal.  

**Funcionalidades:**
- Adicionar receitas e despesas.
- Exibir saldo atual.
- Listar histórico de transações.



---

### 2. Jogo da Adivinhação
Um jogo interativo onde o jogador deve adivinhar um número secreto entre 1 e 100.  

**Funcionalidades:**
- Número secreto gerado aleatoriamente com `Math.random()`.
- Validação de input usando `parseInt()` e verificação de números entre 1 e 100.
- Controle de tentativas com decremento a cada palpite.
- Desenha partes de um boneco a cada erro.
- Indicação se o número secreto é maior ou menor.
- Botão de reiniciar o jogo quando termina.



---

### 3. Sistema de Veículos (POO)
Projeto focado em Programação Orientada a Objetos (POO) utilizando JavaScript.

**Funcionalidades:**
- Classe base `Veiculo`.
- Herança aplicada nas classes `Carro` e `Moto`.
- Encapsulamento do estado do veículo (ligado/desligado).
- Métodos específicos para cada tipo de veículo.
